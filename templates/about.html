About

January 07, 2018

This website is for sharing stories about the television couple "Yağız Egemen" and "Hazan Çamkıran" also known as Yağhaz from the turkish series "Fazilet Hanım ve Kızları". The reason for building this site is because I have seen dedicated fans writing and sharing stories about Yağhaz in twitter and facebook. However, the stories are quickly to get lost and so, to facilitate the reading and sharing of these stories, I made this website.

Rules on this website :
  1. This website is for sharing stories about Yağhaz only
  2. This Website is not for trashing actors
  3. Any stories shared not about Yağhaz, they will be deleted from this website
  4. Any stories that bash actors, will be removed
  5. Any user or comment that's offensive will be removed from this website
What's comming to this website :
  1. User profile
  2. Website supporting different languages
